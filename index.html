Is this legal?

Yes. The bookmarklet reads data already displayed in your own browser — it doesn't access X's servers or API. Sharing blocklists is just sharing your preferences. The install script uses X's own interface to block accounts one by one, the same as doing it manually — just faster.

Will X ban me?

The install script adds short delays between blocks to look like normal usage. That said, blocking hundreds of accounts quickly could theoretically flag automated behavior. We recommend running the script in batches if your list is very large (500+).
